Author

James Spurin is an Engineer focusing on Automation, Containers and Software Development, he is the Author of the best selling Mastering Ansible video course by Packt Publishing. He holds both the CNCF Kubernetes CKA and CKAD qualifications. An advocate for open source software and developer of the Python library xmldataset, featured on the Awesome Python list. Creator and maintainer of the most popular Hexo docker image on Docker Hub. Working within Cloud Engineering in Nomura on the development of a Private Cloud and maintaining an extensive API Development framework written in Python.

He has a gait affecting disability known as HSP. Outside of technology, he spends his time supporting others with the same disability and its similar counterpart (Cerebral Palsy), he is a promoter of a revolutionary surgical treatment (in which he's the 22nd person ever to have it for HSP) known as SDR (Selective Dorsal Rhizotomy).
